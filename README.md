# hexo-theme-pretty

## 介绍

Hexo 配置 butterfly 主题（搭建模板 v1）


## 环境准备

> 基于 hexo-theme-butterfly 4.9.0 master 分支构建

- Node 环境
- Yarn 工具（全局安装）
- Hexo 库（全局安装）


## v1 如何使用

1. 克隆项目

```bash
git clone git@github.com:pokeyaro/hexo-theme-pretty.git
```

2. 安装依赖

```bash
cd hexo-theme-pretty/
yarn
```

3. 渲染启动

```bash
hexo generate
hexo server
```

此时，在浏览器中访问 `http://localhost:4000`，应该能够看到我们的网站了。
